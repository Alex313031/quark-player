About Quark Player

Quark Player is a fork of ElectronPlayer which is itself a fork of (the same author's) Netflix app. It is based on Electron.
I have updated Electron from 12.1.1 to 22.3.9, updated the name and icon, added more streaming services, added more menu commands, and updated npm packages including the Ad Blocker (by Cliqz).
